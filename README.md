# 宁理计算机飞跃计划 (csdiy at NXIST)

本计划面向来自 蜘蛛人 和 TS 两个社团的 2025 级大一新成员以及大二老成员，成功入选本计划的同学将加入属于本计划的小组，并在学长学姐们的带领下，系统性自学国内外优秀的 CS (计算机科学 Computer Science) 课程，并独立完成项目课题。

## 本计划目的？

本计划旨在汇聚具有独立思考、信息检索、问题发现与解决能力的同学，提供一种独立于课内教学、更高质量的培养方案。计算机科学与传统高考科目关联有限，这是一段从零开始的旅程，而其最优质的资源尽在互联网上。与此同时，计算机科学领域许多高含金量的机会并不以背景为门槛。我们期望同学们参与本计划系统性学习计算机科学、未来可以在计算机科学拥有自己的理想信念、确定自己的未来方向、推动创新实践、可以开发出优秀、硬核的项目、甚至直通到世界名校的 PhD 项目，遂诞生本计划。寄望于孵化出一批掌握核心能力与特质的学生，借助这些特质在更高的平台上开拓更广阔的未来。

## 有什么形式？

1. 每周一次讨论与分享活动，集中头脑风暴解决在计划中当前阶段遇到的困难，分享各自的经验
2. 定期举办内部 [Hackathon](https://baike.baidu.com/item/黑客马拉松/8649956)，让小组成员以小队形式在有限时间内开发出一个有趣的项目
3. 组织参与 [GitHub](https://github.com) 上的开源项目，或者创建自己的开源项目，积累实践经验并学习协作开发技能
4. 每阶段末尾举办一次项目成果展示日，所有参与者分享项目成果，展示计划带来的成长
5. 选择方向 **Track** 后，为每位成员匹配一名学长学姐作为**导师**，提供更加个性化的指导

## 如何入选本计划？

放出后规定时间内完成任务书中任务

任务完成过程中**可以**使用 AI 进行文献搜索和思路引导，但请**不要**使用 AI 编写程序或代码补全

- 大一：完成**环境配置任务**和**一个任务书**

- 大二：完成**环境配置任务**和**两个任务书**

### 评分规则

#### 内容完整性

1. 我们在题目中会放出相应的技术路线，并让大家学习相关技术，根据做出来的项目结果对成绩进行打分

2. 其次对于功能的实现我们会进行测试，根据对功能的实现情况进行评分

3. 对于有创新点，或者是更好技术路线想法的同学我们可以进行额外加分

#### 易用性和简洁度

1. 我们在结束后会根据应用功能的使用体验进行评分

2. 我们会查看源码，根据代码的规范性和可读性进行额外加分

3. 我们在结束后会简单测试下性能的利用情况，同等完成度性能更优者可额外加分

#### 情况了解

1. 我们会发放一份在线意愿表，大家可在讨论部分发表自己对本次小组开设的构想，我们会尽力优化本次小组的体验

2. 项目结束后我们会进行一场面试，在面试中会询问同学在 Demo 实现过程中出现的问题和想法，以及自己 Demo 中的一些实现方法和构想，并根据表现进行额外加分(并不是答辩，不需要准备幻灯片)
